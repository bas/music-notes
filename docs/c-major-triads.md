## Practice Tips

1. **Play each chord in sequence** to hear the harmonic progression of the C major scale
2. **Practice moving smoothly** between adjacent chords to build finger dexterity
3. **Notice the patterns** - each chord shares notes with its neighbors
4. **Use these shapes** as building blocks for creating bass lines and walking bass patterns
5. **Experiment with rhythm** - try playing these chords in different rhythmic patterns

## Chord Progressions

Try these common progressions using the triads above:

- **I-IV-V-I**: C-F-G-C (basic progression)
- **I-vi-IV-V**: C-Am-F-G (popular progression)
- **ii-V-I**: Dm-G-C (jazz progression)
- **I-V-vi-IV**: C-G-Am-F (modern pop progression)
